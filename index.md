# RateMyTool(s)

> A community-driven review platform for UH Manoa students to discover and rate online learning resources

## The Problem

UH Manoa students are constantly searching for effective online learning resources to supplement their courseworkâ€”from video tutorials and practice platforms to AI tools and study aids. However, there is no centralized, student-driven platform where they can discover which tools actually work.

## The Solution

RateMyTool(s) is a community-driven review platform specifically for UH Manoa students to rate and review online learning resources, AI tools, and educational platforms. Similar to RateMyProfessor but for digital learning tools.

### Key Features

- **Search by Course**: Find tools by course code (e.g., ICS 314)
- **Category Browsing**: Explore coding practice, math tutoring, AI assistants, and more
- **Student Reviews**: Read detailed reviews from fellow UH Manoa students
- **Personal Dashboard**: Save favorite tools and get personalized recommendations
- **Course Tags**: See which courses each tool works best for

## Use Cases

1. **New Student Discovery**: Find top-rated tools for specific courses
2. **AI Tool Comparison**: Compare ChatGPT vs Claude for coursework
3. **Contributing Back**: Share experiences with tools for specific courses
4. **Major-Specific Recommendations**: Discover trending tools among your major
5. **Course Preparation**: Preview recommended tools before registration
